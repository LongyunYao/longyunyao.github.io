西安

  • DAY8(6-14) 城墙外的大小雁塔、陕西历史博物馆一带
  • DAY9(6-15) 兵马俑、华清池
  • DAY10(6-16) 城墙（南门）及城墙内景点（钟楼，鼓楼，回民街）

路线规划

DAY1(6-7) 塔尔寺

行程规划

早9点出机场，11点到市中心，吃饭然后去市中心找房间放行李。下午坐车去西塔寺，晚上7点半散场。回西宁市中心过夜。

如果有时间可以去千姿湖(黄河清湿地公园)。（天下黄河贵德清）

参考攻略： 西宁塔尔寺夏季营业时间下午几点不允许进入？

公交路线

西宁曹家堡机场 -> (机场大巴市中心线) -> 银龙宾馆 -> (步行) -> 管理站公交站 -> 塔尔寺公交专线 -> 塔尔寺

机场大巴市中心线路径：https://goo.gl/maps/rtx81pfSXgM2

时间开销

  • 早上9点出机场
  • 机场 -> 管理站公交站： 2h 参考价 21 块
  • 管理站公交站 -> 塔尔寺： 1h 参考价 5 块
  • 晚上回西宁住一晚

参考攻略：在西宁，早起去塔尔寺玩半天够吗？只有半天时间

经济开销

塔尔寺 80/人

注意事项

  • 面积比较大，自带水
  • 许多地方不允许拍照

DAY2(6-8) 金沙湾 - （二郎剑景区(151基地)） - 黑马河日落

行程规划

上午提车，途径湟源县、海晏县进入抵达金沙湾景区；下午途径青海湖环线抵达黑马河看夕阳。（青海湖环线时间充裕的话可以考虑二郎剑景区(151基地)，但是据说这个景区一般，或者直接在环线路边拍拍拍）晚上在黑马河过夜。

这段行程是不走日月山和倒淌河的！也不经过拉脊山！是另一条路线

行驶路线

西宁 -> 湟源县 -> 海晏县 -> 金沙湾：https://goo.gl/maps/Q3BZBLonaPu

金沙湾->黑马河：https://goo.gl/maps/19Xw9DQwNB72

时间开销

  • 早上9点左右提车，回宾馆取行李退房
  • 上午赶去金沙湾：2h 可能途径高速，参考价10块
  • 下午经过青海湖环线到达黑马河：2.5h 可能途径高速，参考价未知

经济开销

  • 金沙滩联票100元，里面所有的项目任玩，不联票就是30元一人
  • 二郎剑景区（如果进去的话）旺季（4月-10月）：100元
  • 黑马河不收费

注意事项

  • 可以准备颜色鲜艳一点的衣服准备沙漠中的拍照
  • 注意防晒（墨镜+披肩）
  • 车上记得备水
  • 黑马河可以考虑在湖边住一晚上帐篷，方便第二天看日出
  • 黑马河那里一定一定一定要提前预定，提前预定100左右，现场预定500:)

DAY3(6-9)黑马河看日出- 茶卡盐湖-可鲁克湖-托素湖自然保护区 - 大柴旦 - （水上雅丹，会绕路但是很好看） -（翡翠湖）

行程规划

早起看黑马河日出（剪影），中午茶卡盐湖拍+吃，下午去可鲁克湖-托素湖自然保护区，时间足够就去翡翠湖，晚上之前抵达大柴旦镇。晚上在大柴旦镇过夜。

行车路线

黑马河 -> （途径橡皮山或G109） -> 茶卡盐湖：google地图没有路径，百度和高德有地图

茶卡盐湖 -> （途径外星人遗址） -> 可鲁克湖

可鲁克湖 -> 大柴旦：https://goo.gl/maps/4duciLsKjtE2

时间开销

  • 根据太阳升起时间决定起床时间，预计8点看完出发
  • 早晨橡皮山或G109赶去茶卡盐湖，1.5h，可能收费，参考价15，停留至中午，吃饭 *下午去可鲁克湖-托素湖自然保护区，停留一段时间，去往大柴旦
  • 如果时间充裕，下午可以去翡翠湖看一下
  • 晚上回到大柴旦镇住宿

经济开销

  • 门票70元/人
  • 茶卡盐湖小火车单程50/人
  • 可鲁克湖门票20元/人
  • 翡翠湖不收费

注意事项

  • 今天开车时间比较长
  • 茶卡盐湖可以租硅胶鞋套，很便宜
  • 中午反射效果最好，不过反射光线较强，一定要戴上墨镜并注意防晒。
  • 茶卡盐湖中有一些小黑洞一定不能踩，否则会连鞋带脚一起陷下去
  • 去托素湖一定要从外星人遗址的高速出口下，这边去托素湖的风景非常棒
  • 翡翠湖其实也是盐湖，不过没有经过旅游开发
  • 这条路线还有一种选择，跳过某些景点，转去水上雅丹！！！真的很好看，没开发过不收费

DAY4(6-10) 大柴旦 - 玉门关 - 汉长城、河仓城 - 雅丹地质公园（魔鬼城） - 敦煌

行程规划

早起沿G3011前往甘肃，在中午之前到达玉门关；然后一路到达雅丹地质公园；晚上回到敦煌市区，（夜市可以考虑月牙泉小镇，古代西北的文化+特色美食）

行车路线

大柴旦镇 -> 白色戈壁公路（G3011） -> 党河口附近左转 -> 玉门关

时间开销

  • 早上8点左右开始赶车，大柴旦至玉门关预计4h45min， 中午1点左右到玉门关
  • 玉门关(20分钟)，河仓城(15分钟)
  • 驾车10分钟 汉长城遗址(20分钟)
  • 驾车10分钟 雅丹国家地质公园（魔鬼城）(1小时)
  • 下午4点左右离开魔鬼城驱车前往敦煌市 2h
  • 晚上可以去月牙泉小镇

经济开销

  • 玉门关、河仓城、汉长城的联票为40元
  • 魔鬼城全价票120元/人（门票50元，车票70元），优惠票95元/人（门票25元/人，车票70元/人）

注意事项

  • 出发前做好应急准备

    • 应急食物，尤其是要多买些巧克力，因为巧克力是高热量食物
    • 水是必备的，特别是纯净水，不但人喝，车子有的时候也要使用。比如发动机冷却水缺乏，纯净水可以应急。长距离下坡刹车过热，可以使用纯净水降温等。
    • 有条件的话，检查一下。常规检查机油、冷却液、助力油、玻璃水，重点检查轮胎、备胎胎压和轮胎磨损是不是正常，轮胎上有没有扎钉子。看看千斤顶和更换轮胎的工具是不是在车上，带一个轮胎充气泵也许有用处。有可能的话去4S店使用计算机检查一下发动机、刹车、ABS和电路情况。

  • 无人区白色戈壁公路贼好看，可以停下来拍照片

  • 中途可以慢一点，在魔鬼城等日落会很好看

DAY5(6-11) 敦煌 - 敦煌莫高窟 - 鸣沙山月牙泉 - 敦煌

行程规划

  • 早上9点莫高窟
  • 下午鸣沙山+月牙泉

行车路线

  • 早上从驱车前往莫高窟，可能要稍微早起一点。
  • 下午前往鸣沙山月牙泉
  • 沙洲夜市或者月牙泉小镇夜市

时间开销

  • 早上9点莫高窟 2h
  • 回酒店休息，下午3点左右驱车鸣沙山，游玩两小时至18点等日落
  • 步行至月牙泉看夜景

经济开销

  • 莫高窟 100/人 （已购票）
  • 鸣沙山月牙泉 旺季（5月1日—10月31日）120元/人，学生凭有效学生证件半价优惠
  • 鸣沙山其他服务参考价：骆驼80元/匹；滑翔机160元-1600元/人，以飞行的路程决定；滑沙15元/人；租鞋套10元/副；挑选骑骆驼和滑沙时工作人员拍的照片，20元/张

注意事项

  • 莫高窟景点记得记得记得带上身份证和学生证
  • 月牙泉可以等到9点天黑闭馆，晚上开灯以后很好看
  • 除了七彩丹霞的日落，七彩丹霞日出的景色也特别美。很多大环线的行程中都会安排看七彩丹霞的日落，住宿在七彩镇，出景区时可以办理第二天免费入场，不需要额外付费，因此，何不趁此机会看看不逊色于黑马河日出的七彩丹霞日出景象呢。
  • 敦煌大街小巷的美食中，最出名的便是驴肉面黄与杏皮水，其他的可以在沙洲夜市或者其他地方搜罗一下（其实可以在其他地方找找，因为感觉沙洲夜市会比较载客而且会比较商业化）

DAY6(6-12) 敦煌 - （瓜州稍作调整） - 嘉峪关关城 - 张掖七彩丹霞（日落） - 张掖

行程规划

  • 主要任务是赶车，上午敦煌驾车至瓜州修整，再驾车至嘉峪关
  • 下午参观嘉峪关（如果时间不够就可以考虑舍去，因为绑定的两个景点并不好看，本身关卡也很坑）
  • 接着去七彩丹霞

行驶路线

敦煌市 -> （沿着G3011） -> 瓜州县：https://goo.gl/maps/F6fNxMt5YxH2 走高速，参考收费22元

瓜州县 -> （沿着G312） -> 嘉峪关：https://goo.gl/maps/24kimp21DcR2 走高速，参考收费75元

嘉峪关 -> （沿着G30） -> 临泽县 -> 张掖七彩丹霞：https://goo.gl/maps/ZDUkG31ueF92（仅包含部分线路） 参考收费56元

时间开销

  • 尽早爬起来准备赶车保证当天时间充沛
  • 敦煌->瓜州：2小时
  • 瓜州->嘉峪关:驾车 2.5小时
  • 嘉峪关->七彩丹霞：驾车 2.5小时

经济开销

  • 嘉峪关门票(联票且只可买联票，包含长城博物馆、长城第一墩和悬臂长城)，旺季（5月1日-10月31日）120元/人，学生票半价
  • 张掖的七彩丹霞地貌景区分为临泽县入口（西入口）和肃南县入口（东入口）两个入口，为统一购票，门票54元/人

注意事项

  • 嘉峪关在一些游记中，给人留下的印象并不好，如果时间紧张可以不去或者只是在外城（外城据说不用票都可以进）
  • 七彩丹霞景区好像有一些比较骚的操作：六月去张掖看丹霞哪个时间段最佳呢？
  • 七彩丹霞的日落和日出都很好看，可以尝试骚操作一下第二天早上爬起来看日出

DAY7(6-13) 张掖 - 张掖七彩丹霞（日出） - 祁连山祁连大草原 - 门源油菜花（可能没开门） - 远眺岗什卡雪峰 - 黑泉水库 - 西宁

行程规划

  • 早起看日出（如果拿到允许第二天进入的票的话）
  • 祁连山草原
  • 门源油菜花
  • 黑泉水库
  • 赶回西宁

行车路线

张掖 -> （西张线） -> 祁连山草原

祁连山草原 -> 门源油菜花

门源油菜花 -> 黑泉水库

时间开销

  • 张掖丹霞地质公园(1.5小时)
  • 张掖->祁连 驾车3小时，大草原游玩45min左右
  • 祁连->门源油菜花 驾车1个钟。门源游玩45min左右
  • 门源->宁西 驾车1小时40分钟 途径黑泉水库(30分钟)

经济开销

  • 祁连山草原 免费
  • 门源油菜花 门票免费，观花台收费30元/人。
  • 黑泉水库免费

注意事项

  • 晚上9点之前感去机场还车

DAY8(6-14) 西安 大小雁塔（大慈恩寺、北广场）、陕西历史博物馆一带

行程规划

中午下机，先赶回市内，在城墙外部分景点玩一下，如大小雁塔（北广场、大慈恩寺）、陕西历史博物馆。具体游玩顺序根据酒店位置而定。

路程规划

小雁塔：乘坐21路、402路等公交在小雁塔站下车即到。 西安博物馆：市内搭乘游7路、18路、21路或者地铁2号线在南稍门站下车，再往西走200米就可看到西安博物院。 大雁塔：乘坐21路 、521路等公交车在大雁塔站下车。

时间开销

  • 景点就在市内，交通工具时间应该在半小时以内。
  • 小雁塔游玩时间在1-1.5h
  • 西安博物馆时间在2h
  • 大雁塔游玩时间在1-2h

经济开销

  • 小雁塔免费不免票，凭身份证领取；登小雁塔30元。开放时间：8:00-17:30
  • 西安博物馆限额领票参观（需携带身份证）；每天向社会公众发放免费参观券3000张，发票时间为每个开放日的9:00-16:00
  • 大雁塔北广场免费，开放时间: 8:00-17:30
  • 大慈恩寺50元，大雁塔登塔30元。开放时间：8:00-17:30

注意事项

  • ·大雁塔北广场的音乐喷泉在夜晚游客较多，最佳观景地点要提前占位,建议提前30分钟前往。

DAY9(6-15) 西安 华清池、兵马俑、（秦皇陵地宫）、（秦始皇陵）

行程规划

  • 去往兵马俑时，沿途会经过华清池、秦皇陵地宫和始皇陵，可以考虑上午华清池，下午兵马俑。（兵马俑市内景点下午不会晒）
  • 可以考虑美团参团买套票

时间开销

  • 华清宫旅游时间预计1-2h
  • 兵马俑博物馆旅游时间预计2-3h

经济开销

  • 华清宫旺季150元/人次
  • 兵马俑旺季150元/人次（包含始皇陵的套票）
  • 秦陵地宫参考价35元/人次

注意事项

  • 兵马俑可以请导游讲解或者选择自动讲解机，或者使用微信导游
  • 今天旅游在当地的美团上报一个团，兵马俑+华清宫+地宫+半坡博物馆，导游和讲解都有了，80块费用，如果是学生，需要再付205的门票费（学生票半价），什么都包括了，比自己去方便些
  • 秦陵地宫并不是真正的地宫，而是依据大量文献记载、考古发现和民间传说所提供的资料，运用声、光、电等技术和雕塑、美术、古建等多种手法，模拟而成的秦始皇帝陵地面陵区和地宫上下两层的形式，准确说来，秦陵地宫景区的全称应该是秦陵地宫展览馆。
  • 秦始皇兵马俑博物馆和秦始皇陵门票实行淡、旺季浮动价格，“一票制”管理；观众往返秦始皇兵马俑博物馆和秦始皇帝陵两景区参观，可免费乘坐旅游专车

DAY10(6-16) 西安 城墙南门，钟楼，鼓楼，回民街

行程规划

  • 城墙一共14km，可以骑车绕城墙一圈，但是会比较颠簸
  • 半圈可以南门上，北门下（或者反向）
  • 还可以从南门上，再选一个就近的门下

参考行程：陕西历史博物馆+古城墙+钟鼓楼/回民街+大雁塔喷泉广场西安深度一日游（ 精品小团 三环内接待 景点全涵盖）

时间开销

  • 相对自由，但是注意飞机晚上10点起飞，需要预留时间

经济开销

  • 钟楼和鼓楼的门票均为35元/人
  • 城墙成人54元，1.2-1.4米儿童、学生证(硕士学历以上学生证、国外院校学生证除外)和65-70岁老年人27元。
  • 城墙上可租赁自行车骑行游览，单人自行车45元，双人自行车90元，押金200元。

关于预算

这篇预算是根据第一篇参考自驾游计算的，实际会有出入，我已经懒得算钱了

  1. 机票： 广州-西宁，来回1440/人
  2. 租车： 350*6=2100
  3. 住宿： 2000元/2人 （基本200左右一天，一般连锁酒店在网上订会便宜，民宿什么的我会直接打电话砍价或者到了再定。七月初我们去的时候住宿还好不是很紧张，七月中以后就要另当别论了。）
  4. 油费 ： 800 左右 （青海开车根本不用开空调超省油！）
  5. 过路费： 350 左右 （我们只有兰州到西宁和张掖到兰州开的高速，分别花了80和180的过路费，其他省道国道都是一次性收十块八块的过路费）
  6. 门票： 塔尔寺80/人，茶卡 50/人，玉门关、河仓城、汉长城的联票为40元/人，魔鬼城95元/人，卓尔山70/人，莫高窟 100/人 （已购票），嘉峪关60元/人，门源万亩油菜花60/人，张掖丹霞地质公园74/人，大慈恩寺50元/人，西安day2美团抱团285元/人，钟鼓楼70元/人，城墙54元/人。有地方用了学生证，总花费大概1000元。
  7. 餐饮： 1500左右/2人 （每天都起的挺晚的，一般是吃一天两顿的节奏）
